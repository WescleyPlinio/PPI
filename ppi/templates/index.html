O que são projetos integradores?

Um projeto integrador é uma atividade que conecta diferentes áreas de conhecimento, permitindo que você aplique o que aprendeu na prática para resolver problemas reais de forma mais completa e colaborativa.

O que é este site?

A Plataforma dos Projetos Integradores é um sistema que trás em suas mãos todos os projetos integradores do campus SPP já criados, permitindo que você possa visualizá-los.
